# Social Media Bot - 社交媒体自动运营工具

[English](#english) | [中文](#中文)

## 中文

Twitter/Instagram自动运营工具 - 发帖、点赞、关注、评论。

### 功能
- 自动发布内容
- 批量点赞关注
- 内容排程发布
- 关键词监控自动回复
- 多账号管理

### 安装
```bash
npm install
node index.js
```
